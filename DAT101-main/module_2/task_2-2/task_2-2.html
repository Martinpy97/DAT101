    2. const skal man bruke når man vet at variabelen ikke skal endres i ettertid og let kan man bruke der det er meningen at den skal endres i ettertid.
    Man burde bruke const vanligvis, frem til du lager en variabel du vet skal endres. For å unngå at du endrer variabler med uhell senere.

  2. How do parentheses affect the arithmetic outcome of an expression?

    3. Parenteser bestemmer rekkefølgen på operasjoner i et uttrykk og er derfor viktig å bruke riktig for å få riktig resultat.

  3. How do we limit floating-point numbers to a given number of decimals?

    4. Man kan bruke "toFixed();" for å formatere et tall til spesifisert antall desimaler.
    For eksempel: let tall = 123.1234
    let formatertTall = tall.toFixed(2); = 123.12

    på 0.000001. Vi må se på differansen og se om den er innenfor grensen på akseptverdien.
    ((12.2 - 12.199999) < 0.01) og hvis det er True, så sier vi at tallene er like.

  5. What do comparison operators return, and how can we check if they are FALSE or TRUE?

    6. Sammenligningsoperatorer returnerer en ''boolean value'' som er (True or False) basert på resultat av input.
    Eksempel på dette kan være: "5 == 2 = False" fordi tallene er ikke like og "5 != 2 = True" fordi tallene er ulike.

  6. How does the Math.random() function work, and what can it be used for?

    7. Math.random() genererer tilfeldige desimaltall mellom 0 og 1 og kan brukes i forskjellige ting som for eksempel spill eller simuleringer.
